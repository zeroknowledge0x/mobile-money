# SEP-31: Cross-Border Payments

:::info
See the full [SEP-31 Implementation](https://github.com/sublime247/mobile-money/blob/main/docs/SEP31_IMPLEMENTATION.md) in the repository.

## Overview

SEP-31 enables direct cross-border payments between anchors without requiring the user to have a Stellar account.
